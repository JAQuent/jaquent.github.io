Lab meeting

Nov 17, 2017 by Alex Quent
Tags: lab meeting schemaVR
Categories: experiments

Today I met with my supervisor and we also had lab meeting.

  • We have discussed the difficulty to counter-balance the expectedness of the objects and their locations in a true factorial design. For instance, it is nearly impossible to pair objects so that the expected location of one object is the unexpected location of the other object.
  • We are interested in how the expectedness of an object and its location is related to memory and whether it is possible to get a u-shaped function. Therefore, we decided to measure the expectancy for each object in order to be able to analyse memory performance as a function of the expectedness of both the object and the location.
  • To do this, I will have people rate the expectedness of an object in a room and in 20 possible locations in that room. Starting with kitchen, I will create a MATLAB script which can be paused so and which everyone can run at home or at their workplace.
  • Our aim for the final experiment is to draw stimuli from the ratings matrix to get a nice sample of congruent, neutral and incongruent objects. For that, I should create a java version so that I am able to put it on https://www.prolific.ac/.
  • For the current pilot experiment, we decided to have the ratings at the end so they don’t affect subsequent memory decisions such as the retrieval of the location.
